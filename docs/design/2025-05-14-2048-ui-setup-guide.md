# 2048 游戏 UI 场景创建指导手册

## 准备工作

1. 打开 Cocos Creator，加载 `Game2048` 场景
2. 在层级管理器中找到并删除以下旧节点（如果存在）：
   - `settingsPanel` 及其所有子节点
   - `gameOverPanel` 及其所有子节点

---

## 第一部分：创建设置面板 (settingsPanel)

### 步骤 1：创建根节点
1. 在 **Canvas** 节点上右键 → **创建空节点**
2. 命名：`settingsPanel`
3. 位置：(0, 0, 0)
4. 大小：720 × 1280
5. 取消勾选 **Active**（默认隐藏）

### 步骤 2：创建背景
1. 在 `settingsPanel` 上右键 → **创建 2D 对象 → Sprite（精灵）**
2. 命名：`bg`
3. 位置：(0, 0, 0)
4. 大小：720 × 1280
5. 颜色：#000000（黑色）
6. 透明度：180

### 步骤 3：创建内容容器
1. 在 `settingsPanel` 上右键 → **创建空节点**
2. 命名：`content`
3. 位置：(0, 0, 0)
4. 大小：560 × 400
5. 添加组件：**Sprite**
6. Sprite 颜色：#FFFFFF（白色）
7. （可选）圆角：如有 9 宫格资源可设置圆角

### 步骤 4：创建标题
1. 在 `content` 上右键 → **创建 2D 对象 → Label（标签）**
2. 命名：`titleLabel`
3. 位置：(0, 140, 0)
4. 文本：`游戏设置`
5. 字号：48
6. 颜色：#333333
7. 水平对齐：居中

### 步骤 5：创建"网格大小"标签
1. 在 `content` 上右键 → **创建 2D 对象 → Label**
2. 命名：`gridSizeLabel`
3. 位置：(0, 60, 0)
4. 文本：`网格大小`
5. 字号：32
6. 颜色：#666666
7. 水平对齐：居中

### 步骤 6：创建数值显示
1. 在 `content` 上右键 → **创建 2D 对象 → Label**
2. 命名：`gridSizeValue`
3. 位置：(0, 10, 0)
4. 文本：`4`（默认显示 4）
5. 字号：72
6. 颜色：#8F7A66（主题色）
7. 勾选：**Bold**（粗体）
8. 水平对齐：居中

### 步骤 7：创建滑条容器
1. 在 `content` 上右键 → **创建空节点**
2. 命名：`sliderContainer`
3. 位置：(0, -60, 0)
4. 大小：400 × 80

### 步骤 8：创建最小值标签
1. 在 `sliderContainer` 上右键 → **创建 2D 对象 → Label**
2. 命名：`minLabel`
3. 位置：(-180, 0, 0)
4. 文本：`4`
5. 字号：28
6. 颜色：#999999

### 步骤 9：创建滑条（关键步骤）
1. 在 `sliderContainer` 上右键 → **创建 UI 组件 → Slider（滑条）**
2. 命名：`slider`
3. 位置：(0, 0, 0)
4. 大小：300 × 20
5. 选中 `slider` 节点，在属性检查器中设置：
   - **Handle**（滑块）：保持默认或自定义图片
   - **ProgressBar**（进度条）：保持默认
6. 展开 **Slider** 组件：
   - **Progress**: 0
   - **SlideEvents**: 清空（代码中会绑定）

#### 设置滑条样式：
1. 找到 `slider` 的子节点 `Handle`：
   - 大小：40 × 40
   - 颜色：#8F7A66（主题色）
2. 找到 `slider` 的子节点 `Bar` 或 `ProgressBar`：
   - 颜色：#8F7A66（主题色）

### 步骤 10：创建最大值标签
1. 在 `sliderContainer` 上右键 → **创建 2D 对象 → Label**
2. 命名：`maxLabel`
3. 位置：(180, 0, 0)
4. 文本：`8`
5. 字号：28
6. 颜色：#999999

### 步骤 11：创建按钮容器
1. 在 `content` 上右键 → **创建空节点**
2. 命名：`buttonContainer`
3. 位置：(0, -160, 0)
4. 大小：480 × 80

### 步骤 12：创建取消按钮
1. 在 `buttonContainer` 上右键 → **创建 UI 组件 → Button**
2. 命名：`cancelButton`
3. 位置：(-120, 0, 0)
4. 大小：200 × 70
5. 选中按钮，设置 **Sprite** 组件颜色：#CCCCCC
6. 找到子节点 `Label`：
   - 文本：`取消`
   - 字号：32
   - 颜色：#FFFFFF

### 步骤 13：创建开始游戏按钮
1. 在 `buttonContainer` 上右键 → **创建 UI 组件 → Button**
2. 命名：`startButton`
3. 位置：(120, 0, 0)
4. 大小：200 × 70
5. 选中按钮，设置 **Sprite** 组件颜色：#8F7A66（主题色）
6. 找到子节点 `Label`：
   - 文本：`开始游戏`
   - 字号：32
   - 颜色：#FFFFFF

### 步骤 14：绑定 SettingsPanel 脚本
1. 选中 `settingsPanel` 节点
2. 在属性检查器中点击 **添加组件 → 自定义脚本 → SettingsPanel**
3. 绑定以下属性：
   | 属性名 | 绑定到节点 |
   |--------|-----------|
   | Grid Size Slider | sliderContainer/slider |
   | Grid Size Value Label | content/gridSizeValue |
   | Start Button | buttonContainer/startButton |
   | Cancel Button | buttonContainer/cancelButton |

- `scoreManager` ✓
- `inputHandler` ✓

## 常见问题排查

### 问题 1：滑条拖动时数值不更新
**检查**：
- `gridSizeSlider` 是否正确绑定到 Slider 组件（不是节点）
- `gridSizeValueLabel` 是否正确绑定到 Label 组件

### 问题 2：游戏结束面板不显示
**检查**：
- `gameOverPanelComponent` 是否正确绑定到 gameOverPanel 节点
- gameOverPanel 节点上是否有 GameOverPanel 脚本组件

### 问题 3：按钮点击无响应
**检查**：
- 按钮的 **Button** 组件是否正常
- 按钮的子节点 Label 是否遮挡了点击区域
- 脚本中的事件绑定是否正确
