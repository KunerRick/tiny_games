## 1. 游戏概述

2048 是一款经典的数字益智游戏，玩家通过滑动屏幕移动方块，使相同数字的方块合并，最终目标是合成 2048 方块。

---

## 2. 核心玩法

### 2.1 基本规则
- **网格**: 4x4 ~ 8x8 可选（默认 4x4）
- **初始**: 随机生成两个方块（90%概率为2，10%概率为4）
- **操作**: 上下左右滑动移动所有方块
- **合并**: 相同数字的方块碰撞时合并（2+2=4, 4+4=8...）
- **生成**: 每次移动后在空位随机生成一个新方块
- **目标**: 合成 2048 方块，达到后可继续挑战更高数字
- **结束**: 当没有可移动方向时游戏结束

### 2.2 计分规则
- 每次合并获得分数：合并后的数值（如 2+2=4，得4分）
- 实时显示当前分数
- 记录历史最高分（按网格大小分别记录）

## 7. 核心算法

### 7.1 移动和合并逻辑

```typescript
// 以向左移动为例
function moveLeft(tiles: TileData[], gridSize: number): MoveResult {
    const newTiles: TileData[] = [];
    let scoreGained = 0;
    
    for (let row = 0; row < gridSize; row++) {
        // 获取该行的所有方块
        const rowTiles = tiles.filter(t => t.row === row).sort((a, b) => a.col - b.col);
        
        // 合并相同数字的方块
        const merged: TileData[] = [];
        let col = 0;
        
        for (let i = 0; i < rowTiles.length; i++) {
            const current = rowTiles[i];
            const next = rowTiles[i + 1];
            
            if (next && current.value === next.value) {
                // 合并
                merged.push({
                    ...current,
                    col: col,
                    value: current.value * 2,
                    isMerged: true
                });
                scoreGained += current.value * 2;
                i++; // 跳过下一个
            } else {
                // 不合并，只移动
                merged.push({
                    ...current,
                    col: col
                });
            }
            col++;
        }
        
        newTiles.push(...merged);
    }
    
    return { tiles: newTiles, scoreGained, hasMoved: tilesChanged(tiles, newTiles) };
}
```

### 7.2 生成新方块

```typescript
function spawnTile(tiles: TileData[], gridSize: number): TileData | null {
    // 找出所有空位
    const emptyCells: {row: number, col: number}[] = [];
    
    for (let row = 0; row < gridSize; row++) {
        for (let col = 0; col < gridSize; col++) {
            if (!tiles.some(t => t.row === row && t.col === col)) {
                emptyCells.push({row, col});
            }
        }
    }
    
    if (emptyCells.length === 0) return null;
    
    // 随机选择一个空位
    const cell = emptyCells[Math.floor(Math.random() * emptyCells.length)];
    
    // 90%概率生成2，10%概率生成4
    const value = Math.random() < 0.9 ? 2 : 4;
    
    return {
        id: generateId(),
        row: cell.row,
        col: cell.col,
        value: value,
        isNew: true
    };
}
```

### 7.3 检查游戏结束

```typescript
function checkGameOver(tiles: TileData[], gridSize: number): boolean {
    // 还有空位？
    if (tiles.length < gridSize * gridSize) return false;
    
    // 检查是否有可合并的相邻方块
    for (const tile of tiles) {
        const neighbors = [
            {row: tile.row - 1, col: tile.col},
            {row: tile.row + 1, col: tile.col},
            {row: tile.row, col: tile.col - 1},
            {row: tile.row, col: tile.col + 1}
        ];
        
        for (const n of neighbors) {
            const neighbor = tiles.find(t => t.row === n.row && t.col === n.col);
            if (neighbor && neighbor.value === tile.value) {
                return false; // 可以合并
            }
        }
    }
    
    return true; // 游戏结束
}
```

## 9. 手势识别

### 9.1 滑动检测

```typescript
// 最小滑动距离阈值
const SWIPE_THRESHOLD = 50;

// 在 InputHandler.ts 中
onTouchStart(event: EventTouch) {
    this.startPos = event.getLocation();
}

onTouchEnd(event: EventTouch) {
    const endPos = event.getLocation();
    const deltaX = endPos.x - this.startPos.x;
    const deltaY = endPos.y - this.startPos.y;
    
    // 判断滑动方向
    if (Math.abs(deltaX) > Math.abs(deltaY)) {
        // 水平滑动
        if (Math.abs(deltaX) > SWIPE_THRESHOLD) {
            this.onSwipe(deltaX > 0 ? Direction.RIGHT : Direction.LEFT);
        }
    } else {
        // 垂直滑动
        if (Math.abs(deltaY) > SWIPE_THRESHOLD) {
            this.onSwipe(deltaY > 0 ? Direction.UP : Direction.DOWN);
        }
    }
}
```
